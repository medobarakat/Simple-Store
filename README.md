## Getting Started
To get a local copy of the repository please run the following commands on your terminal:
- ```$ cd <folder>```
- ```$ git clone https://github.com/medobarakat/Simple-Store.git ```
- ```$ cd Simple-Store ```
- ```npm install```
- ```npm start```
